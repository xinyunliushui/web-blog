## 博客以及管理后台
- 该项目的创建使用[Create React App](https://github.com/facebook/create-react-app)，前端功能代码使用cursor生成
- 该项目主要是为了配合后端服务[go-blog](https://github.com/xinyunliushui/go-blog)。

## 本地开发
```bash
# 安装依赖
yarn
# 本地启动，启动成功后会在默认浏览器中打开 http://localhost:3000
yarn start
```


## 说明
- 因为项目中的大部分功能都是通过AI生成，重点关注功能是否实现，因此代码层面的技术方案不具有参考性
